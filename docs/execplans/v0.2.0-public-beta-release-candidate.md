## Release classification

This release remains **Public Beta**. The dashboard and API are suitable for reproducible demonstrations, comparative industry analysis, and portfolio review. The analytical ratios and composite indicators remain experimental and must not be interpreted as credit, insolvency, causal, investment, or policy conclusions.

The candidate is not a `1.0.0` stability declaration. Manual rendered-browser accessibility checks and release-owner acceptance are required before final publication.

## Scope

The release candidate consolidates already-merged capabilities rather than introducing unrelated features:

- Streamlit sample, official Census AIES snapshot, upload, BEA, and Census ASM workflows;
- Explore, Compare, and Scenario Lab analysis;
- canonical `/v1` analytical and metadata API contracts with documented legacy aliases;
- typed source, cache, evaluation, scenario, API, and export lineage;
- privacy-safe `user-upload` provenance;
- CSV, JSON, XLSX, and CSV-lineage companion exports;
- truthful no-auth public-data catalog and typed `/v1/meta/public-data` endpoint;
- protected hosted quality gates and production Docker smoke validation.

Out of scope:

- metric or heuristic formula changes;
- provider expansion;
- dependency churn unrelated to a release blocker;
- API v2 or compatibility changes;
- workflow or Docker redesign;
- promotion to stable `1.0.0`.
